### vite(Vite 是一个 web 开发构建工具，由于其原生 ES 模块导入方式，可以实现闪电般的冷服务器启动。)

> 全局安装 create-vite-app
```
$ cnpm i -g create-vite-app
```

> 创建项目 runoob-vue3-test2

```
create-vite-app runoob-vue3-test2
```

> 运行

```
$ cd runoob-vue3-test2
$ cnpm install
$ cnpm run dev
```
